TROUBLESHOOTING:
═══════════════════════════════════════════════════════════════════════════════

ISSUE: GitHub Pages shows 404 error
FIX: Check Network tab - if CSS/JS files show 404, paths are wrong
    All paths should be relative like "css/styles.css" NOT "/css/styles.css"

ISSUE: Refresh goes back to Dashboard
FIX: Hash routing not working. Check console for errors.
    setupHashRouting() should log hash changes.

ISSUE: URL doesn't change when clicking tabs
FIX: switchTab() might not be updating hash correctly
    Check console: should see [APP] Switched to tab: ... | Hash: #...

ISSUE: Console shows 404 for index.html on refresh
FIX: This is NORMAL on GitHub Pages. GitHub routes all requests to index.html
    This is how GitHub Pages works - it's not an error!

ISSUE: Navigation works on local but not on GitHub Pages
FIX: Check if basePath is detected correctly
    Console should show: IS GitHub Pages: true (for GitHub Pages)
    AND detected basePath should include /SalonElena/
